# TP3 LOG3000

### Nom du projet
Calculatrice Web

### Numéro d’équipe
60

### Objectif
L’objectif est de mettre en place de bonnes pratiques de gestion de versions, de documentation et de tests dans un dépôt collaboratif sur GitHub.

### Prérequis d’installation
- Un compte GitHub
- Git installé localement
- Python 3 installé (3.9 ou supérieur)
- pip installé

### Instructions d’installation
1. Cloner le dépôt
2. Créer un environnement virtuel
3. Installer les dépendances
4. Lancer l’application
